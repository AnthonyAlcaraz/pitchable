    1. Update the root `package.json` to add workspaces configuration:
       - Add `"workspaces": ["apps/*"]`
       - Add Turborepo as a devDependency: `npm install -D turbo`
       - Add root scripts: `"dev": "turbo dev"`, `"build": "turbo build"`, `"lint": "turbo lint"`
       - Keep ALL existing dependencies and scripts (the backend runs from root)

    2. Create `turbo.json` at root:
       ```json
       {
         "$schema": "https://turbo.build/schema.json",
         "pipeline": {
           "dev": { "cache": false, "persistent": true },
           "build": { "dependsOn": ["^build"], "outputs": ["dist/**"] },
           "lint": {}
         }
       }
       ```

    3. Create `apps/web/` directory and initialize the frontend:
       - Run: `mkdir -p apps/web/src`
       - Create `apps/web/package.json` with:
         - name: "@pitchable/web"
         - dependencies: react@^18, react-dom@^18, react-router-dom@^7, zustand@^5
         - devDependencies: @types/react, @types/react-dom, @vitejs/plugin-react, vite@^6, typescript@^5, tailwindcss@^4, @tailwindcss/vite, autoprefixer
         - scripts: "dev": "vite --port 5173", "build": "tsc -b && vite build", "preview": "vite preview"

    4. Create `apps/web/vite.config.ts`:
       - Import @vitejs/plugin-react and @tailwindcss/vite
       - Configure proxy: `/api` and `/auth` routes proxy to `http://localhost:3000`
       - This allows the frontend to call the backend without CORS issues in dev

    5. Create `apps/web/tsconfig.json` and `apps/web/tsconfig.app.json`:
       - Target ES2020, module ESNext, strict mode
       - Include src directory
       - jsx: "react-jsx"

    6. Create `apps/web/index.html`:
       - Standard Vite HTML template with `<div id="root"></div>` and script tag pointing to `/src/main.tsx`
       - Title: "Pitchable"

    7. Create `apps/web/src/main.tsx`:
       - Standard React 18 createRoot entry point rendering `<App />`

    8. Create `apps/web/src/App.tsx`:
       - Import BrowserRouter from react-router-dom
       - Render a placeholder `<div>Pitchable</div>` (routes added in Plan 06)

    9. Create `apps/web/src/index.css`:
       - Tailwind v4 CSS: `@import "tailwindcss";`

    10. Create `apps/web/src/vite-env.d.ts`:
        - `/// <reference types="vite/client" />`

    11. Run `npm install` from root to install all workspace dependencies.

    IMPORTANT: The existing NestJS backend stays at root level. Do NOT move any backend files. The monorepo structure is:
    ```
    /                   # NestJS backend (existing)
    /apps/web/          # React frontend (new)
    /turbo.json         # Turborepo config
    ```
